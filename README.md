**Essentials**
---
Simple logger for info, error and debug logs.

**Example**
---
    # import library
    from MakeLog import make_log as log
    
    # initialse Class
    logger = log.initialise()
    
    # write info log
    logger.info('info_test')
    
    # write error log
    logger.error('error_test')
    
    # write debug log
    logger.debug('debug_test')
